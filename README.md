Lottery Ticket Hypothesis (LTH):
“There exists a subnetwork—approximately 10% the size of a trained network of size N—that can achieve comparable performance to the full network.”

Strong Lottery Ticket Hypothesis (SLTH):
“There exists a subnetwork—approximately 10% the size of an untrained network of size N—that can achieve comparable performance to the fully trained network if trained in isolation.”

Until now, no consistent method has existed to reliably identify these subnetworks.
We present a framework that uncovers the lottery ticket every time.
